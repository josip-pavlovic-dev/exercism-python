# Exercism Python Track

This repository contains my solutions to [Exercism Python Track](https://exercism.org/tracks/python) exercises.

## About

Exercism is a platform for learning programming through practice exercises and mentorship. This repo tracks my progress through the Python track.

## Structure

Each exercise is in its own folder:

- `hello-world/` - Introduction exercise
- More exercises will be added as I progress through the track

## How to Use

1. Download exercises using Exercism CLI:

   ```bash
   exercism download --track=python --exercise=<exercise-name>
   ```

2. Run tests locally:

   ```bash
   cd <exercise-folder>
   pytest
   ```

3. Submit solution:
   ```bash
   exercism submit <filename.py>
   ```
